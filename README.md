# Brain-MRI-Image-Enhancement
โปรเจคนี้มีวัตถุประสงค์เพื่อศึกษาวิธีการพัฒนาและเพิ่มคุณภาพรูปภาพของสมองที่สร้างภาพด้วยคลื่นแม่เหล็กไฟฟ้า โดยใช้ภาษา Python และเครื่องมือต่าง ๆ เช่น OpenCV, DIPY ซึ่งโปรเจ็กต์นี้เป็นส่วนหนึ่งของรายวิชา Digital Image Processing

# Tools 
**Language:** Python  
**Libraries:** OpenCV (cv2) , DIPY, NumPy, stackview
